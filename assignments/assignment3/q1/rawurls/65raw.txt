			<p>The CEO of The&nbsp;<a href="/companies/the-coca-cola-company_id256">Coca-Cola Co</a> has said the company's bottling system in China has evolved to the point where re-franchising is the "next logical step".</p>

<p>Speaking to analysts following the company's <a class="internalLink" href="/news/coca-cola-co-bids-farewell-to-transition-2015-with-4-sales-dip_id119353.aspx">full-year and Q4 results announcement yesterday</a>, Muhtar Kent said that Coca-Cola's bottling partners in the country were "ready to take on additional territory".</p>

<p>The group&nbsp;<a class="internalLink" href="/news/coca-cola-co-ramps-up-re-franchising-programme_id119352.aspx">also announced yesterday that it intends to re-franchise</a> 100% of its North America bottling territories by the end of 2017, including all cold-fill production facilities. At the same time, the company said it had entered into a non-binding letter of intent to refranchise company-owned bottling operations in China, to existing partners China Foods, which is part of COFCO, and Swire Beverage Holdings.</p>

<p>Kent said the announcements, along with the pending creation of <a class="internalLink" href="/news/coca-cola-enterprises-secures-germany-iberia-merger_id117751.aspx">Coca-Cola European Partners</a> and <a class="internalLink" href="/news/the-coca-cola-co-to-greenlight-anheuser-busch-inbev-as-africa-partner-analyst_id119100.aspx">Coca-Cola Beverages Africa</a>, as well as the company's investment in its Indonesian bottler, would strengthen the global bottling system.</p>

<p>"To put it in perspective, adjusting for these transactions, the percent of our 2015 volumes sold through company-owned bottlers would have decreased from 18% to 3%," he said.</p>

<p>China saw a 1% volume increase for the company in 2015, with&nbsp;president &amp; COO&nbsp;James Quincey&nbsp;describing the environment in the country as "slowing".&nbsp;</p>

<p>"We're looking to do better in 2016," he added.</p>

        function getCookie(name) {
            var dc = document.cookie;
            var prefix = name + "=";
            var begin = dc.indexOf("; " + prefix);
            if (begin == -1) {
                begin = dc.indexOf(prefix);
                if (begin != 0) return null;
            }
            else {
                begin += 2;
                var end = document.cookie.indexOf(";", begin);
                if (end == -1) {
                    end = dc.length;
                }
            }
            return unescape(dc.substring(begin + prefix.length, end));
