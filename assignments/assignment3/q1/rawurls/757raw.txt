        <h2 class="newsTitle">『相棒』の視聴率低迷　「反町戦犯説」は本当か　</h2>
        <p class="hbody">　水谷豊（63）主演『相棒 season14』（テレビ朝日系）が思うように視聴率が伸びていない。現在放送中のシーズンは新相棒に反町隆史（42）を迎え、初回18.4％と上々のスタートを切ったものの、2話以降それを超えることはできず、昨年12月9日放送の第8話では2007年のseason6以来、8年ぶりに12％台を記録。元日スペシャルの第10話では16.7％と盛り返したが、1月13日の第11話では12.6％と今シーズン最低を記録している。その後13.7％、15.3％、14.6％とやや持ち直したものの、昨年3月まで放送されたseason13の平均視聴率17.4％と比べると低迷ぶりは明らか。一部メディアからは「反町が原因ではないか」という声もチラホラ…。果たして「反町戦犯説」は本当なのか。(NEWS ポストセブン)</p><a class="newsLink" href="http://www.news-postseven.com/search?q=%E5%8F%8D%E7%94%BA%E6%88%A6%E7%8A%AF%E8%AA%AC%20%E8%A6%96%E8%81%B4%E7%8E%87%E4%BD%8E%E8%BF%B7%20%E7%9B%B8%E6%A3%92" onmousedown="this.href='http://www.news-postseven.com/search?q=%E5%8F%8D%E7%94%BA%E6%88%A6%E7%8A%AF%E8%AA%AC%20%E8%A6%96%E8%81%B4%E7%8E%87%E4%BD%8E%E8%BF%B7%20%E7%9B%B8%E6%A3%92'">[NEWS ポストセブンで関連記事を探す]</a>    </div>

$(function(){
    $('div.headline').each(function(){
        var $div = $(this);
        var $hide = $('.headlinePic',this);
        var $img = $('img',this);
        var $url = $img.attr('data-src');

        if ($url === void 0) {
            return;
        }

        var imgObj = new Image();
        $(imgObj).one('error',function(){
            $hide.remove();
            $div.removeClass("hasImg");
        });
        $(imgObj).one('load',function(){
            var w=imgObj.width,h=imgObj.height,dw=$img.attr('data-width'),dh=$img.attr('data-height');
            if(w > dw || h > dh) {
                if(w/dw > h/dh) $img.css('width',dw+'px');
                else $img.css('height',dh+'px');
            }
            $img.attr('src', $url);
        });
        imgObj.src = $url;
    });

    $('li.article a div.imgWrap').each(function(){
        var $div = $(this);
        var $img = $('img', this);
        var $url = $img.attr('data-src');

        if ($url === void 0) {
            return;
        }

        var imgObj = new Image();
        $(imgObj).one('error',function(){
            $div.remove();
        });

        $(imgObj).one('load',function(){
            $img.attr('src', $url);
        });
        imgObj.src = $url;
    });
});
</script>
