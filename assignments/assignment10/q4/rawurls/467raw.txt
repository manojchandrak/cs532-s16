			<p>The Chapel Hill Public Library is planning to add four to six hours to it&#8217;s current schedule.</p>
<p>Before deciding when these hours would be, the library is asking for <a href="https://docs.google.com/forms/d/1FNa3OsOMZ1s0X2W9SPTkpDxZzSBcZiHbYmaRl9dbvLs/viewform?platform=hootsuite" target="_blank">public input</a> to help figure out which hours would best serve the community.</p>
<p>The deadline to submit input is February 15.</p>
<p>According to the library website, these expanding hours will come with no increase to the current budget.</p>
<p>The library will announce its expanded hours in March.</p>
