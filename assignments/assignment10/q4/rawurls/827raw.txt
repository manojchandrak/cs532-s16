<br />
全国に店舗を持つベーカリー「アンデルセン」。その丸井今井札幌店（札幌市中央区）で、かわいいパンを見つけました。「札幌ごまあんぱん」（税込162円）です。<br />
<br />
こんがり焼かれたまあるいパンの表面には、「札幌」の焼印。サイドには白と黒のゴマをたっぷりとまぶしてあります。どことなくレトロな風合いがかわいい！ 中身のあんこは、つぶあんです。ゴマの香ばしさとほっくり甘いあんこ、間違いない組み合わせ。<br />
<br />
実はこの商品、&ldquo;道産素材&rdquo;を使ったシリーズの一環で、北海道産あんこが使われているのだとか。店頭にはほかにも、北海道産の小麦粉でつくった塩パンなどが並んでいましたよ。<br />
<br />
レギュラー商品として販売中とのこと。奥にはカフェスペースもあるので、イートインでもいただけます。カジュアルな手土産にも良いかもしれません。<br />
<br />
観光だとデパートや百貨店に行くことはなかなかないと思いますが、ふらっと歩いてみると素敵な出会いがあるものです。<br />

				var proportion = $this.data( 'proportion' );
				var	w = $this.attr('width');
				var actual_w = $this.width();

				if ( !proportion ) {
					proportion = $this.attr('height') / w;
					$this.data( 'proportion', proportion );
				}

				if ( actual_w != w ) {
					$this.css( 'height', Math.round( actual_w * proportion ) + 'px' );
				}
			});
